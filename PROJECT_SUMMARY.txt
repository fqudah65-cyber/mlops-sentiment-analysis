================================================================================

includes a RESTful API, authentication, containerization, CI/CD automation,
comprehensive testing, and detailed documentation.

GETTING STARTED
================================================================================

1. INSTALL DEPENDENCIES
   pip install -r backend/requirements.txt

2. RUN APPLICATION
   cd backend/src
   python app.py

3. RUN TESTS
   pytest backend/tests/ -v

4. BUILD DOCKER IMAGE
   docker build -f docker/Dockerfile -t sentiment-api:latest .

5. RUN DOCKER CONTAINER
   docker run -p 5000:5000 sentiment-api:latest

6. ACCESS API
   http://localhost:5000/health
